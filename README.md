## 🎮 Features

* ⚔️ System walki (atak, blok, hitboxy)
* 🤖 AI przeciwników (różne poziomy trudności)
* 🏹 Ataki dystansowe (łuk)
* ❤️ System zdrowia i mikstur
* 🎵 Dźwięki i muzyka
* 🗺️ Wiele map
* 👥 Tryby:

  * Singleplayer (vs AI)
  * Multiplayer (1v1)

---

## 🧱 Project Structure

```
main.py              # game loop + state machine
characters.py        # characters & combat system
items.py             # arrows, potions
assets_manager.py    # sounds & assets loading
config.py            # settings & constants
ui.py                # UI (menu, HUD)
map.py               # maps & platforms
```
```
mistral_doc          # Dokumentacja projektu zgadnij kto ją napisał
```
